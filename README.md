# GrowClimate Website

Professional B2B climate control equipment website for agriculture industry.

## 🌐 Live Site

**Production**: https://growclimate.com  
**CMS Admin**: https://growclimate.com/admin

## 🚀 Tech Stack

- **Frontend**: Vue 3 + Vite 5 + Tailwind CSS 3
- **Routing**: Vue Router 4 (History mode)
- **Deployment**: Netlify
- **CMS**: Netlify Identity + Git Gateway

## 📁 Project Structure

```
growclimate-site/
├── src/
│   ├── components/     # Vue components
│   ├── views/          # Page components
│   ├── data/           # Static data files
│   ├── router/         # Vue Router config
│   └── App.vue         # Root component
├── public/             # Static assets
├── static/admin/       # Netlify CMS files
├── dist/               # Production build (auto-generated)
├── netlify.toml        # Netlify deployment config
└── package.json
```

## 🛠️ Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🌿 Product Categories

- **Dehumidifiers**: RYCM-50C/90C/138C (Compressor), RY-100/200 (Desiccant)
- **Humidifiers**: UH-3L/6L (Ultrasonic), EH-12L (Evaporative)
- **Ventilation**: AC-3HP Industrial Air Conditioner

## 🌱 Target Industries

- Greenhouse / Horticulture
- Hydroponics & Vertical Farming
- Mushroom Cultivation
- Seed Storage & Drying
- Fruit Drying Facilities

## 📝 Content Management

The site uses **Netlify CMS** for content management. To access the CMS:

1. Visit `/admin` on your deployed site
2. Log in with your Netlify Identity account
3. You can edit:
   - Products (Dehumidifiers, Humidifiers, Ventilation)
   - Applications/Use Cases
   - Blog Articles
   - Case Studies
   - Team Members
   - Certifications

## 🔗 Deployment

This site is deployed on **Netlify** with GitHub integration.

1. Push code to GitHub
2. Netlify automatically builds and deploys
3. Enable Netlify Identity for CMS access
